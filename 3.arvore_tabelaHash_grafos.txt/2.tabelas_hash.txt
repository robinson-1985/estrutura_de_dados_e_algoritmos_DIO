                    TABELA DE ESPALHAMENTO

Uma tabela hash, de dispersão ou espalhamento é uma estrutura de dados especial,
que associa chaves de pesquisa a valores.

                        HASHING

Uma tabela hash é uma generalização da idéia de array, porém utiliza uma função
denominada Hashing para espalhar os elementos, fazendo com que os mesmos fiquem
de forma ordenada dentro do "array" que define a tabela.

                    Por que espalhar?

A tabela hash permite a associação de "valores" a "chaves";

Valores: é a posição ou índice onde o elemento se encontra;

Chave: parte da informação que compõem o elemento a ser manipulado.

Espalhar facilita a busca na estrutura de dados, pois a partir de uma chave podemos 
acessar de forma rápida uma posição do "array".
